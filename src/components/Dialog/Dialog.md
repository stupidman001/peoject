参数支持：
title	    对话框标题	                    string	     提示
width	    宽度	                       string	     50%
top	        与顶部的距离	                string	      15vh
visible     是否显示dialog（支持sync修饰符） boolean	   false


事件支持：
opened	模态框显示事件
closed	模态框关闭事件

插槽说明：
default	    dialog的内容
title	    dialog的标题
footer	    dialog的底部操作区
